# Wallpaper Engine Workshop Downloader GUI

A simple tool to download Wallpaper Engine workshop items with a modern interface.

This is a reworked version of the original project with an updated GUI and small improvements.

## Credits  

Original project by SteamAutoCracks  
https://github.com/SteamAutoCracks/WallpaperEngineWorkshopDownloader  

## Features  

- Download workshop items without using your own Steam account  
- Clean and minimal GUI  
- Easy to use  

## Dependencies  

- https://github.com/oureveryday/DepotDownloaderMod  
- .NET 8.0 Runtime  

## Usage  

1. Install .NET 8.0 Runtime  
   https://aka.ms/dotnet-core-applaunch?framework=Microsoft.NETCore.App&framework_version=8.0.0&arch=x64&rid=win10-x64  

2. Open the Wallpaper Engine workshop  
   https://steamcommunity.com/app/431960/workshop/  

3. Copy the link of the item you want  
   Example:  
   https://steamcommunity.com/sharedfiles/filedetails/?id=1234567890  

4. Run the application  

5. Paste the link into the app  

6. Select your Wallpaper Engine folder  
   It must contain the folder `projects/myprojects`  

7. Click Download  

## Notes  

- Make sure the selected path is correct  
- Some items may fail depending on availability
